# Published LinkedIn Posts

Archive of all published LinkedIn posts.

## File Naming Convention

`YYYY-MM-DD_[brief-topic].md`

Example: `2026-07-15_water-heater-strapping.md`

## What to Save

- The full post text as published
- Date published
- Any notable performance data (optional but valuable)

Saving published posts creates a searchable archive that prevents repetition and reveals what topics have been covered.
