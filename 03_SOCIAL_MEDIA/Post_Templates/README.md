# Post Templates

Reusable post formats and frameworks. When a post structure works well, save it here so it can be used again.

---

## How to Use

Copy a template, fill in the specifics, and adapt to the current topic. Templates save time and maintain consistency.

## How to Save a New Template

When a post format performs well or proves reusable:
1. Strip out the specific topic/content
2. Replace with placeholder text in [brackets]
3. Save as `Template_[description].md`
4. Note what the template is good for at the top of the file
