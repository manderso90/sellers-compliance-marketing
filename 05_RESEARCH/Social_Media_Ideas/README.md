# Social Media Ideas

A running capture of content ideas that aren't scheduled yet.

## How to Use

When an idea comes up — from a job, a client question, a trend, a competitor post — log it here before it's forgotten. Ideas don't need to be polished; they just need to be captured.

## Format

Add ideas as bullet points or short notes. When an idea is developed into a draft, move it to the appropriate `Drafts/` folder.
