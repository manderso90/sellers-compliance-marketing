## Adding New Projects

When starting a new project:
1. Add a section above.
2. Define the goal clearly.
3. List key tasks with checkboxes.
4. Link to any relevant strategy documents.
5. Assign an owner and target date.

When a project is complete, move it to `09_ARCHIVE/` with a summary of outcomes.
