## How to Use Commands

When you type a command, Claude will:
1. Read the corresponding skill file for instructions
2. Check the content library for relevant existing material
3. Follow the defined process and produce the output
4. Ask clarifying questions if needed

Always provide as much context as possible with a command. Instead of `/linkedin-post`, try `/linkedin-post about water heater strapping for real estate agents`.
