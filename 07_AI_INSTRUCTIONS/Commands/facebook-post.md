# Command: /facebook-post

**Usage:** `/facebook-post [topic or description]`

**Skill:** `07_AI_INSTRUCTIONS/Skills/Create_Facebook_Post.md`

---

## What It Does

Drafts a Facebook post for Seller's Compliance targeting homeowners and sellers. Follows homeowner-friendly tone and platform guidelines automatically.

## Examples

```
/facebook-post about what sellers need to know before listing
/facebook-post using before/after photo from Pete
/facebook-post reassuring homeowners that compliance is simple
/facebook-post about low-flow toilets
```

## Output

A complete Facebook post draft including hook, body, and CTA — ready for review. Saved to `03_SOCIAL_MEDIA/Facebook/Drafts/`.
