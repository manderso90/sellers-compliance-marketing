# Command: /repurpose-content

**Usage:** `/repurpose-content [paste content or describe source]`

**Skill:** `07_AI_INSTRUCTIONS/Skills/Repurpose_Content.md`

---

## What It Does

Takes an existing piece of content and adapts it for a different platform, format, or audience. Rewrites for native fit rather than copy-pasting.

## Examples

```
/repurpose-content [paste LinkedIn post] → Facebook
/repurpose-content [paste Facebook post] → LinkedIn for agents
/repurpose-content turn these 5 tips into a checklist post
/repurpose-content adapt this educational post into an FAQ entry
```

## Output

A rewritten draft ready for the target platform, saved to the appropriate `Drafts/` folder.
