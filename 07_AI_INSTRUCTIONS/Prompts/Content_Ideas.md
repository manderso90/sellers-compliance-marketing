# Prompt: Content Ideas

Use this prompt when you need a fresh batch of content ideas — for a specific platform, pillar, or topic area.

---

## Prompt Template

```
Generate [NUMBER] content ideas for Seller's Compliance.

Platform: [LinkedIn / Facebook / Both]
Content Pillar: [Education / Social Proof / Industry Insight / Behind the Scenes / Reassurance / Local / Any]
Topic area (optional): [e.g., water heater strapping, smoke detectors, working with realtors]
Audience: [Real estate agents / Homeowners / Both]

For each idea, provide:
- Post hook (first line)
- Content pillar
- Platform fit
- Why this would resonate with the audience

Also flag any ideas that would benefit from photos from Pete.

Check `04_CONTENT_LIBRARY/Compliance_Tips.md` and `03_SOCIAL_MEDIA/*/Published/` first to avoid repeating recent topics.
```

---

## When to Use

- Starting a new content calendar
- Feeling stuck on what to post
- Brainstorming for a specific service or audience
- Filling gaps in the content pillar mix
