# Social Media Assistant — Operating Instructions

Use this file when assisting with any social media task. It covers both LinkedIn and Facebook.

For platform-specific instructions, see:
- `07_AI_INSTRUCTIONS/LINKEDIN_ASSISTANT.md`
- `07_AI_INSTRUCTIONS/FACEBOOK_ASSISTANT.md`

---

## Before Writing Any Post

1. **Identify the platform** — LinkedIn and Facebook have different audiences and tones.
2. **Identify the audience** — Which ICP is this post for? (See `00_COMPANY_CONTEXT/Ideal_Customer_Profiles.md`)
3. **Identify the content pillar** — Education / Social Proof / Industry Insight / Behind the Scenes / Reassurance / Local
4. **Check the content library** — `04_CONTENT_LIBRARY/` — is there existing material to build from?
5. **Check the content calendar** — `03_SOCIAL_MEDIA/Content_Calendar/` — what's already planned?

---

## Universal Content Standards

- **Educational content over promotional content.** A post that teaches is more valuable than a post that sells.
- **Be specific.** "California requires dual water heater straps" beats "compliance matters."
- **Be accurate.** Never invent compliance rules. Reference documented requirements only.
- **Be concise.** Real estate professionals and homeowners are busy.
- **No fear tactics.** Inform, don't alarm.
- **No legal conclusions.** We explain what's required; we don't give legal advice.

---

## Content Pillars

| Pillar | Description | Best For |
|---|---|---|
| Education | Teach compliance facts | Both platforms |
| Social Proof | Testimonials, completed jobs | Both platforms |
| Industry Insight | Real estate market + compliance | LinkedIn |
| Behind the Scenes | Field work, team, process | Facebook |
| Reassurance | Make compliance feel manageable | Facebook |
| Local | LA/OC specific content | Both platforms |

---

## Saving Content

- Drafts: save to the appropriate `Drafts/` folder
- Published: save to the appropriate `Published/` folder with the publish date in the filename
- Templates: save reusable formats to `03_SOCIAL_MEDIA/Post_Templates/`

---

## When You Finish a Post

Ask:
- Is this accurate? (No invented compliance rules)
- Is this appropriate for the platform?
- Is this appropriate for the identified audience?
- Is there a clear next step for the reader?
- Does this reflect the brand voice?
