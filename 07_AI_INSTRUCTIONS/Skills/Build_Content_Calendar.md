# Skill: Build Content Calendar

**Triggered by:** `/content-plan [month]`

---

## Purpose

Build a monthly social media content calendar for Seller's Compliance covering both LinkedIn and Facebook.

---

## Inputs

- **Month/year** (e.g., July 2026)
- **Any upcoming priorities** (campaigns, events, seasonal relevance)
- **Any content to avoid** (topics recently covered — check `03_SOCIAL_MEDIA/LinkedIn/Published/` and `03_SOCIAL_MEDIA/Facebook/Published/`)

---

## Process

1. **Review what's already been published.** Check `Published/` folders for both platforms. Don't repeat recent topics.

2. **Check current priorities.** Read `00_START_HERE/CURRENT_PRIORITIES.md`. Align content with active goals.

3. **Check the content library.** Review `04_CONTENT_LIBRARY/Compliance_Tips.md` and `04_CONTENT_LIBRARY/Educational_Content/` for material to draw from.

4. **Plan by content pillar.** Distribute across pillars:
   - LinkedIn: Education, Social Proof, Industry Insight, Behind the Scenes
   - Facebook: Education, Reassurance, Local Relevance, Social Proof

5. **Build the calendar.** Use the template in `03_SOCIAL_MEDIA/Content_Calendar_Template.md`.
   - LinkedIn: 3–5 posts per week
   - Facebook: 3–4 posts per week
   - Mix platforms so the same topic doesn't run on both on the same day

6. **Flag gaps.** Note any weeks where fresh photos from Pete would improve the plan.

7. **Output the completed calendar** ready to save as `03_SOCIAL_MEDIA/Content_Calendar/YYYY-MM_Content_Calendar.md`.

---

## Output Format

- Completed monthly calendar table (date / platform / pillar / topic / status)
- Brief rationale for any non-obvious choices
- List of any items that need Pete's photos or other inputs before drafting
