# Action Items

Outstanding tasks and follow-ups.

## Format

Save action items as `YYYY-MM-DD_Action_Items.md` or maintain a rolling file `Open_Action_Items.md`.

Each item should include:
- [ ] Task description
- **Owner:**
- **Due:**
- **Status:** Open / In Progress / Done

Move completed items to the relevant meeting notes or archive rather than deleting them — completed tasks are a record of progress.
