# Orientation Notes

This document is the starting point for anyone new to the Seller's Compliance marketing workspace — human or AI.

---

## What This Repository Is

This repository is the marketing operating system for Seller's Compliance. It contains:

- Company background and brand context
- Service documentation
- Social media strategies and content
- Research and competitive intelligence
- Active project tracking
- AI instructions and workflows
- Institutional knowledge and lessons learned

Everything is organized to make it easy to pick up where someone else left off.

---

## Getting Oriented Quickly

**If you're a new human team member:**
1. Start with `README.md` in the root — it explains everything at a high level.
2. Read `00_COMPANY_CONTEXT/` to understand who we are.
3. Read `01_COMPANY_KNOWLEDGE/Services_Overview.md` to understand what we do.
4. Check `06_PROJECTS/Active_Projects.md` to see what's in progress.
5. Review `07_AI_INSTRUCTIONS/` to understand how AI tools are used here.

**If you're an AI assistant (Claude or similar):**
1. Read `CLAUDE.md` in the root directory first. It has everything you need to know to start working.
2. Check `06_PROJECTS/Active_Projects.md` before starting a new task.
3. Check `04_CONTENT_LIBRARY/` before creating new content.
